# Telegram Client for Android

A localized Android client for Telegram, built on the official open-source codebase.

## Features

- **Chinese Interface**: Complete Simplified Chinese localization
- **Network Optimization**: Built-in connectivity for seamless access
- **Ready-to-Use Accounts**: Supports direct login without SMS verification hassles
- **Official Foundation**: Based on Telegram Android 12.5.1 source code

## Download

Get the latest release from our [Releases page](https://github.com/telecn/tg/releases).

## Installation

1. Download the APK file from Releases
2. Enable "Install from Unknown Sources" in your device settings
3. Install and launch the application
